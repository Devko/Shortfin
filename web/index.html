Shortfin

An extremely fast web server

What is it?

Shortfin is a very fast and lightweight open-source web server that serves static content only.
Take a look at the benchmarks or download and install it to see how fast it is.

Soon it will also be a reverse proxy and http accelerator.

Platforms

Shortfin is completely written in C and compiles and runs under Linux
but will be ported to OpenBSD, FreeBSD and Solaris in the near future.
